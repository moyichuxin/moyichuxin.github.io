Ubuntu 22安装与优化

发表于 2024-04-23 分类于 💻技术 ， Ubuntu 阅读次数：

在多次安装Ubuntu后，我发现安装完成后的优化步骤基本一致。但因为很长时间才会用到一次，所以经常会忘记具体步骤，需要重新查找资料。现在，我把相关资料进行了总结，并整理在这个文档中，以便未来不时之需。

1.基本更新

  • 在软件和更新中的附加驱动选择NVIDIA驱动，一般选择（专有，tested）
  • 软件更新器更新驱动
  • 设置里检查蓝牙、wifi、声音、屏幕拓展功能

2.clash安装

  • 下载地址

  • 把安装包移动至合适文件夹

  • 解压：tar -zxvf Clash.for.Windows-0.20.37-x64-linux.tar.gz

  • 在对应文件夹终端： ./cfw

  • 设置开机自启动，加入代理机场链接即可

  • 系统设置中网络——>网络代理——>手动http和https代理均为127.0.0.1。端口号都是7890。忽略主机默认：localhost, 127.0.0.0/8, ::1
